重命名

```powershell
hostnamectl set-hostname node1.itcast.cn
hostnamectl set-hostname node2.itcast.cn
hostnamectl set-hostname node3.itcast.cn
```

IP配置

```powershell
[ipv4]
method=manual
address1=192.168.159.138/24
dns=8.8.8.8
gateway=192.168.159.2

```

重启网卡

```powershell
 systemctl restart NetworkManager
```

host文件互绑

```powershell
192.168.159.128 node1 node1.itcast.cn
192.168.159.132 node2 node2.itcast.cn
192.168.159.133 node3 node3.itcast.cn

```

防火墙以及Selinux关闭

```powershell
 systemctl stop firewalld
 systemctl disable firewalld
 setenforce 0

```

```powershell
vim  /etc/selinux/config
SELINUX=disabled
#/c 前面条件足执行后面条件
sed -i '/SELINUX=enforcing/cSELINUX=disabled' 
```

基础软件安装

```powershell
dnf -y install vim wget rsync -y 
```
